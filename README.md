# Spotify Login Clone 🎵

A **Spotify login page clone** built with pure **HTML5 and CSS3**. Faithfully replicates Spotify's login interface including the brand identity, form layout, dark background, green accents, and overall visual design.

---

## 📋 Description

This project is a static recreation of Spotify's login page. The goal was to practice pixel-accurate UI replication using only HTML and CSS — no JavaScript or frameworks. It focuses on form layout, Spotify's brand colors (black background, #1DB954 green), and clean typographic hierarchy.

---

## ✨ Features

- 🟢 **Spotify Branding** — Matches Spotify's black/dark theme with green accent color (#1DB954)
- 📋 **Login Form** — Email and password fields with Spotify-styled buttons
- 🖼️ **Logo** — Spotify logo integrated in the header
- 🎨 **Pure CSS** — All visual effects done without JavaScript
- 📐 **Centered Layout** — Flexbox-centered login card
